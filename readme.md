# Презентация по теме "Комплексный анализ"

## Задание
Разработать презентацию на тему "Комплексный анализ" с использованием фреймворка Reveal.js.

## Требования к презентации:
- Использование Markdown там, где это возможно
- Минимальное количество слайдов: 20
- Наличие титульного и заключительного слайдов
- Использование слайдов-разделителей перед каждым разделом
- Наличие меню для быстрого перехода к слайду
- Наличие навигации (стрелки, номера слайдов)
- Оформление в отдельном CSS файле
- Читаемое оформление (соответствующие цветовые сочетания)

## Обязательные элементы:
1. Определение комплексных чисел
2. Формы записи комплексных чисел (алгебраическая, тригонометрическая, показательная)
3. Примеры каждой формы записи
4. Программа на JavaScript для конвертации между формами записи
5. Комплексная плоскость и геометрическое представление
6. Программа для построения вектора на комплексной плоскости
7. Операции с комплексными числами (сложение, вычитание, умножение, деление)
8. Программа для выполнения операций с комплексными числами

## Технические требования:
- Использование фреймворка Reveal.js
- Подключение библиотек math.js и KaTeX
- Отдельный файл стилей
- Наличие всех обязательных плагинов
